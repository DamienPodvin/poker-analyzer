INSTALLATION RAPIDE
====================

WINDOWS :
1. Copie tous les fichiers essentiels
2. Double-clic sur install.bat
3. Édite config.json
4. Double-clic sur lancer.bat

MAC/LINUX :
1. Copie tous les fichiers essentiels
2. Terminal : chmod +x install.sh lancer.sh
3. Terminal : ./install.sh
4. Édite config.json
5. Terminal : ./lancer.sh
